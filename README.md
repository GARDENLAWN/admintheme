# GardenLawn Admintheme

This is the custom admin theme for the GardenLawn project.

## Overview

This theme customizes the Magento admin panel interface to match the GardenLawn branding and improve usability for administrators.

## Directory Structure

- **Magento_Backend**: Customizations for the Magento Backend module.
- **etc**: Configuration files.
- **web**: Web assets (CSS, JS, images).
